# 🚀 Needs 平台 - 后台一键启动说明

## 概述

本文档说明如何快速启动后台服务，用于测试 Flutter App。

---

## 📋 系统要求

### 必需软件
- **PHP** >= 8.1
- **MySQL** >= 8.0
- **Redis** >= 5.0
- **Composer**（PHP 依赖管理）

### 安装检查

```bash
# 检查 PHP
php -v

# 检查 MySQL
mysql -V

# 检查 Redis
redis-cli --version

# 检查 Composer
composer --version
```

---

## 🎯 快速启动

### macOS / Linux

#### 方式 1：直接运行脚本（推荐）

```bash
cd /Users/cuixiaocun/Desktop/needs
chmod +x start-backend.sh
./start-backend.sh
```

**如果需要运行数据库迁移：**
```bash
./start-backend.sh --migrate
```

#### 方式 2：手动启动

```bash
# 1. 进入项目目录
cd /Users/cuixiaocun/Desktop/needs/needs-api

# 2. 启动 MySQL（如果未运行）
brew services start mysql  # macOS
# 或
sudo systemctl start mysql  # Linux

# 3. 启动 Redis
brew services start redis  # macOS
# 或
redis-server  # Linux

# 4. 启动 Laravel 开发服务器
php artisan serve --host=0.0.0.0 --port=8000
```

### Windows

#### 方式 1：双击运行批处理

直接双击 `start-backend.bat` 文件

**如果需要运行数据库迁移：**
```cmd
start-backend.bat --migrate
```

#### 方式 2：命令行运行

```cmd
cd C:\Users\<YourUsername>\Desktop\needs
start-backend.bat
```

## 📱 App 配置

在 Flutter App 中，API 已配置为连接到本地后台：

```dart
// lib/config/app_config.dart
static const String API_BASE_URL = 'http://localhost:8000/api';
```

### 如果使用物理设备或模拟器

如果 App 运行在真实手机上，需要改为你的电脑 IP：

```bash
# 获取你的电脑 IP（macOS / Linux）
ifconfig | grep "inet " | grep -v 127.0.0.1

# 获取你的电脑 IP（Windows）
ipconfig
```

然后在 App 中改为：
```dart
static const String API_BASE_URL = 'http://192.168.1.100:8000/api';  // 改为你的 IP
```

## 🔧 常见问题

### Q1: Port 8000 已被占用

如果端口 8000 已被使用，可以改为其他端口：

```bash
php artisan serve --host=0.0.0.0 --port=8001
```

### Q2: MySQL 连接失败

检查 MySQL 配置：

```bash
# 编辑 .env 文件
cat needs-api/.env | grep DB_

# 或检查 MySQL 服务状态
sudo systemctl status mysql  # Linux
brew services list | grep mysql  # macOS
```

### Q3: Redis 连接失败

确保 Redis 已启动：

```bash
# 检查 Redis 是否运行
redis-cli ping
# 应该返回 PONG

# 如果返回 Connection refused，启动 Redis
redis-server  # 前台运行
# 或
redis-server --daemonize yes  # 后台运行
```

### Q4: 依赖安装失败

清除缓存并重新安装：

```bash
cd needs-api
rm -rf vendor composer.lock
composer install --no-cache
```

### Q5: "Permission denied" 错误

给脚本执行权限（macOS / Linux）：

```bash
chmod +x start-backend.sh
```

## 🛑 停止服务

按 `Ctrl+C` 停止 Laravel 服务器即可。

如果需要停止其他服务：

```bash
# 停止 MySQL
brew services stop mysql  # macOS
sudo systemctl stop mysql  # Linux

# 停止 Redis
brew services stop redis  # macOS
redis-cli shutdown  # 全平台
```

## 🔐 安全注意

**开发环境使用以下配置：**

```
API_BASE_URL=http://localhost:8000/api
APP_ENV=local
APP_DEBUG=true
ALIPAY_SANDBOX=true
```

**生产环境需要改为：**

```
API_BASE_URL=https://api.example.com/api
APP_ENV=production
APP_DEBUG=false
ALIPAY_SANDBOX=false
```

1. **保持终端打开**：启动后端后，保留该终端窗口打开以便查看日志
