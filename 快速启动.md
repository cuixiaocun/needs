# ⚡ Needs 平台 - 快速启动指南

## 🚀 一键启动（推荐）

### macOS / Linux

```bash
cd /Users/cuixiaocun/Desktop/needs
./start-backend.sh
```

### Windows

双击：`start-backend.bat`

---

## ✅ 启动完成确认

看到以下输出表示成功：

```
==========================================
  API 服务器运行地址: http://localhost:8000
  API 健康检查: http://localhost:8000/api/health
==========================================
```

---

## 📱 启动 Flutter App

在另一个终端中：

```bash
# 进入 App 目录
cd /Users/cuixiaocun/Desktop/needs/.worktrees/flutter-app-dev/needs_app

# 启动 App（iOS）
flutter run -d iphone

# 或启动 App（Android）
flutter run -d emulator-5554
```

## 🧑‍💻 测试流程

1. **启动后台**
   ```bash

   ```

2. **启动 App**
   ```bash
   flutter run
   ```

3. **测试登录**
   - 邮箱：test@example.com
   - 密码：password123

4. **查看日志**
   - 后台日志在启动脚本的终端中显示
   - App 日志使用 `flutter logs`

## 🛑 停止服务

在后台启动脚本的终端中按 `Ctrl+C` 停止

---

## 🔧 常见命令

```bash
# 重新启动数据库迁移
./start-backend.sh --migrate

# 清除应用缓存
cd needs-api && php artisan cache:clear

# 查看实时日志
php artisan logs

# 数据库调试
cd needs-api && php artisan tinker
```

## 💡 开发建议

- **分窗口工作**：后台一个终端，App 一个终端
- **查看日志**：后台终端会显示所有 API 请求
- **快速调试**：使用 `php artisan tinker` 操作数据库
